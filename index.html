CSS stands for Cascading Style Sheets.

  • CSS is the language we use to style an HTML document.
  • CSS describes how HTML elements should be displayed.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.

p5.js is a free and a open-source software.

  • p5.js is a JavaScript library for creative coding, with a focus on making coding accessible and inclusive for artists, designers, educators, beginners, and anyone else!
  • Using the metaphor of a sketch, p5.js has a full set of drawing functionality.
  • You can think of your whole browser page as your sketch, including HTML5 objects for text, input, video, webcam, and sound.
